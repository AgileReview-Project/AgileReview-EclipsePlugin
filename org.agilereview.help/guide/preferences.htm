Preferences

To configure preferences refer to 'Window>Preferences' menu item.

Here the user can configure some settings of the plugin.

Option Description Default
author Default author name. Chooses the OS user name
review source project This is the used review source project. In this project all reviews and comments are stored. You need to know the folder for distributed reviews. Agilereviews
commentcolor Default color for the comments annotated in the editor RGB: 0,255,128
smartsuggestion When enabeling the smartsuggestion some textfields (e.g. the recipient) are filled with the last Input of this field. Enabled
Default template for XLS export The location where the default XLS export template is stored. An examplary template can be found at the Sourceforge project site.  
Default XLS export location The default location where the exported review data is to be stored.
