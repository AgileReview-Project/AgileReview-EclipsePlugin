Review concept

Definition of "review" in context of the AgileReview plugin

A review is a set of information and comments. It can contain multiple files of multiple projects. A review contains the following information:

  • ID - the review's ID (unique)
  • External Reference - a reference to e.g. a bug tracker item
  • Responsibility - the person that is in charge for this review
  • Status - the review's actual status (in progress, finished)
  • Description - a short description of the this review
