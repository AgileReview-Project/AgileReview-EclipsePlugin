Storage concept

All review data is stored in the AgileReview project located in the workspace of Eclipse.

Storage of reviews

The AgileReview project folder contains a sub-folder for each review. This sub-folder contains the "review.xml" that stores the information related to the review.

Storage of comments

Comments are stored on a per- author base in the sub-folder representing the review. For each author a file "author_$AUTHORNAME.xml" is created that stores the authors comments.

Sharing reviews

Reviews can be shared by sharing the AgileReviews project folder e.g. via content versioning systems such as SVN oder git.
ATTENTION: When commenting files, also the files that were commented have to be commited as otherwise the newly created comments will be treated as global comments.
