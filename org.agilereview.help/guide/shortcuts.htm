AgileReview Perspective

Shortcuts Use When Meaning
CTRL + Shift + + Editor or CommentSummary view is active. This shortcut creates a new comment on the cursor position in the editor. After the first creation the cursor is in description field in the detail view.
CTRL + Shift + # Editor is active and cursor in highlighted range of a comment. This shortcut opens the selected comment in the detail view and also selects this comment in the comment summary view. In this case the selected comment is the comment which the cursor position is in.
ALT + Shift + + global This shortcut opens the wizard for inserting a new reply to the comment which is actually open in the detail view.
ALT + Shift + DEL global This shortcut deletes the comment which is actually open in the detail view.
DEL ReviewExplorer or CommentSummary view is active. This shortcut deletes the selected review(s) or comment(s).

Shortcuts Use When Meaning
CTRL + N global This shortcut opens the 'New'-wizard. There it is possible to create a new review.
ALT + Shift + N global This shortcut opens the 'New' context menu. There it is possible to create a new review.
