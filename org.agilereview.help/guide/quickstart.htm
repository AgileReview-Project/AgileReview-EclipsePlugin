Quickstart

The following is a quick start guide to the AgileReview plugin. It is meant to walk you through the basic steps to use the plugin easily.

Short Introduction

This plugin is used to do code reviewing in your development environment. So it is possible to switch between the development perspective and the code-reviewing perspective as quickly as possible. This reduces effort in setting up your review environment.

Reviewing code is done in two steps:

  • creating a review: comments are grouped in reviews, so for commenting code, a review has to be created first (see also: definition of review).
  • commenting code: The AgileReview plugin allows to add/modify/delete code based comments. This means that a comment can be added to a selected range of code. After that the corresponding lines will be highlighted and e.g. comment text, recipient, or the priority can be set. By selecting an already existing comment it's details are shown and can be modified or the selected comment(s) can be deleted.

Using Agilereview

Notice: A cheatsheet for all of the now mentioned actions in the help section is provided.

For all actions it is necessary to open the AgileReview perspective.

Create new Review

  1. Option 1
    1. Go to the ReviewExplorer
    2. Click on the "Create new Review" button, or select "Create new Review" from the context menu. This will open the "New Review" wizard
    3. Insert data for the review (especially the ReviewName can not be empty)
    4. Confirm this data with Finish
  2. Option 2
    1. Open the "New" Menu via CTRL+N or File->New->Other
    2. Select from the category "AgileReview" the entry "Review"
    3. perform steps 3 and 4 of option 1

Now the Review is created and shown in the ReviewExplorer as the active review. Detailed data can be seen and modified in the Detail View.

Change Review Information

  1. Go to the ReviewExplorer
  2. Select the review you want to change. This displays the review information in the detail view.
  3. Go into the detail view
  4. Change the data

All changes will be saved automatically when leaving the detail view.

Delete Review

  1. Go to the ReviewExplorer
  2. Select the review(s) you want to delete
  3. Press DEL key or select "Delete" from the context menu

Create new Comment

  1. If no review is set to "active", activate a review in the ReviewExplorer
  2. Select the code line(s)/section where the comment should be placed. If the filetype is not of type "xml" or "java", the comment will be a "global" comment that is not related to a piece of code, but only to the actually opened file.
  3. Add a comment
    1. Option 1: Click "Add new comment" in the top right corner of the comments summary view
    2. Option 2: Press CTRL+Shift+"+"
  4. The new comment is displayed in the detail view and it's data can be modified

All changes will be saved automatically when leaving the detail view.

Change Comment

  1. Select the comment you want to change via
    1. Option 1: selecting it in the comments summary view
    2. Option 2: pressing CTRL+Shift+"#" in the editor that contains the comment while the cursor is in the highlighted range of the comment.
  2. The comment's details will be shown in the detail view. Data can be changed in this view

All changes will be saved automatically when leaving the detail view.

Delete Comment

  1. Go to the comments summary view
  2. Select the comment(s) you want to delete
  3. Press DEL or the Click "Delete comment(s)" in the top right corner of the comments summary view
