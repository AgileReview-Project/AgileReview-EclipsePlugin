Tips and tricks

The following tips and tricks give some helpful ideas for increasing your productivity. They are divided into the following sections:


Editor   |   Review Explorer   |   Comments Summary View   |   Detail View


Editor

Add comment (Select a range of text and) press CTRL+Shift+"+"
Delete comment Go with the cursor to a range of text that is commented and press CTRL+Shift+"-"
Show comment in detail view Go with the cursor to a range of text that is commented and press CTRL+Shift+"#"

Review Explorer

Create review
  • Press STRG+N, select Review from AgileReview category
  • Open context menu, select "Create new review"
  • Press the "Create new review" button
Delete review
  • Select one or more reviews and press DEL
  • Select one or more reviews and select "Delete" in the context menu
Activate review
  • Press the "Activate selected review" button
  • Select a review and select "Activate selected review" in the context menu
Open/close review
  • Press the "Open/close reviews" button
  • Select one or more reviews and select "Open/close selected reviews" in the context menu
Switch AgileReview source Project
  • Press the "Switch AR source project" button

Comments Summary View

Add comment Press the "Add new comment" button in the top right corner of the view
Delete comment
  • Select one or more comments in the table and press DEL
  • Select one or more comments and press the "Delete comment" button in the top right corner of the view
Show comment in detail view Select a comment in the table via click (show in detail view), double click or by pressing enter (show also in editor)

Detail View

Add reply
  • Press ALT+Shift+"+"
  • Press the "New reply" button in the top right corner of the view
Delete comment
  • Press ALT+Shift+DEL
  • Press the "Delete shown resource" button in the top right corner of the view
Relocate comment
  • Press the "Relocate comment" button in the top right corner of the view
Revert changes Press the "revert" button in the top right corner of the view
