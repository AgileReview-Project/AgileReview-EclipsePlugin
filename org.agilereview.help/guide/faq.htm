Frequently Asked Questions

What are the minimal system requirements to run AgileReview?

The plugin has been tested with the following Eclipse versions:

  • Eclipse 3.4 - Ganymede
  • Eclipse 3.5 - Galileo
  • Eclipse 3.6 - Helios

Why is the project named AgileReview?

The project was named AgileReview as reviews can be done in a fast, flexible way which was summarized in the word agile. This has nothing to do with Agile Software Developement.

Where is the review data saved? How can I share it?

The review data is saved as a project in your Eclipse workspace. It contains all reviews and comments. You can share it via your version control system. Also you can checkout review data of e.g. colleagues to join reviews.

What are Smart Suggestions?

Smart Suggestions provide you with previously entered values for common data fields such as comment recipient.

Why can't I review files using specific editors?

At the moment only single-page text editors (such as Java editor or plain text editor) are supported. Support for other (also multi-page) editors will be included in future releases.

Which template should I use for the review export?

The template should be an instantiation of the provided raw templates in the download section. You can implement some more analytical functionalities for your export template. The only constraint is, that the first four excel sheets should not be changed to guaranty a successful export with AgileReview.
